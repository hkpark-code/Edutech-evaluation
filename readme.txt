================================================================================
                공교육 중심 에듀테크 기업 가치평가 웹사이트
================================================================================

■ 프로젝트 개요
--------------------------------------------------------------------------------
본 프로젝트는 공교육 분야 에듀테크 기업의 가치를 체계적으로 평가하기 위한 
웹 기반 자가진단 시스템입니다. 기본 자격, 기업 역량, 공교육 가치의 3개 영역에 
걸쳐 다양한 평가 지표를 통해 기업의 종합적인 가치를 산출합니다.

※ 본 자가진단 결과는 1회용 진단으로 응답하신 데이터와 진단 결과는 
   누적 관리되지 않습니다.


■ 주요 기능
--------------------------------------------------------------------------------

1. 기업 정보 입력
   - 기업명 입력
   - 비즈니스 모델(BM) 선택 (복수 선택 가능)
     * 플랫폼형, 콘텐츠형, 디바이스형, 서비스형, 네트워크형
   - 복수 BM 선택 시 비율 설정 (매출액/정규직원 기준)
   - 기업 단계 선택 (초기/성장/성숙)

2. 기본 자격 평가 (Pass/Fail)
   - 5개 기본 항목 + 2개 디바이스 추가 항목에 대한 Pass/Fail 평가
     [기본 항목 - 5개]
     * 사업자 등록 증빙
     * 개인정보 처리방침
     * 서비스 가용성
     * 교육 윤리 준수
     * 데이터 투명성
     [디바이스 BM 추가 항목 - 2개]
     * 전자파 적합성 (디바이스 BM 선택 시에만 표시)
     * 유해물질 규제 (디바이스 BM 선택 시에만 표시)
   - 하나라도 Fail 시 다음 단계 평가는 진행되지만 최종 결과는 Fail 처리

3. 기업 역량 평가 (40점 만점)
   - 재무 안전성 (23.3%)
     [초기 기업 - 택1 선택]
     * 현금흐름 안전성 또는 이자보상비율 중 1개 선택
       → 미선택 지표의 가중치는 다른 지표에 비례 배분
     [공통 항목]
     * 부채비율
     * 유동비율
     * 영업이익률
     [자금조달역량 - 택1 선택] (초기 기업만)
     * 투자금액 규모 또는 투자기관 신뢰성 중 1개 선택
       → 미선택 지표의 가중치는 다른 지표에 비례 배분
     [투자 유치 실적 - 선택사항] (성장/성숙 기업)
     * 투자금액 규모, 투자기관 신뢰성 (평가 포함 여부 선택 가능)
   - 운영 지속성 (33.3%)
     * 고객 유지율
     * 매출액 성장률
   - 기술 신뢰성 (43.4%)
     * 품질 모니터링 체계
     * 개선 프로세스 운영
     * 장애 대응 체계
     * 사후 관리 체계
     * 정보보호 인증 (ISMS-P, ISO/IEC 27001 등)

4. 공교육 가치 평가 (60점 만점)
   - 교육적 가치 (48.9%)
     * 효과성: 사용자 참여도, 학습 성과 개선, 학습자 주도성, 교사 업무 효율화, 시스템 편의성
     * 적합성: 국가교육과정 연계성, 교육정책 부합도, 현장 활용도 (도입기관/이용자 택1), 재계약률, 교육 형평성
     * 혁신성: AI 기반 맞춤형 학습, 혁신적 학습 환경, 다양한 학습 형태 지원
     * 전문성: 교육 전문 인력 확보, 교육 전문 인력 비율
   - 기술적 가치 (24.4%)
     * 공통 기술: R&D 역량, 기술적 차별성, 학습데이터 활용(BM 연동), 
                  학습경험 설계(사용성/접근성/몰입형), 기술 확장성(시스템 연계/데이터 이관), 해외시장 확장 가능성
     * BM별 역량: 전문 인력 규모, 핵심 스킬셋 보유(BM별 연동), 기술 성과물(BM별 연동)
   - 사회적 가치 (26.7%)
     * 사회적 책임: 저사양 기기 지원, 오프라인 모드 지원, 데이터 절감 기능
     * 윤리 준수: 데이터 및 AI 윤리 책임
     * 협력생태계 구축: 교육기관 파트너십

5. 평가 결과
   - 회사명, 기본자격 충족 여부, 종합 등급 및 해설
   - 기업역량/공교육가치 종합 점수 및 세부 영역 점수 (도넛 파이차트)
     * 달성률(%) - 원 중앙 표시
     * 획득 점수 / 만점 - 상단 표시
   - 종합 제언 (영역별 구분)
     * 기업 역량 분석 - 강점/개선 필요 항목
     * 공교육 가치 분석 (공통 영역) - 강점/개선 필요 항목
     * BM별 차별화 영역 분석 - 선택한 BM별 강점/개선 필요 항목
     * 종합 전략 제언 - 등급에 따른 맞춤형 전략
   - 등급 판정 (AAA/AA/A/BBB/BB/B/CCC/CC/C/D)
   - PDF 인쇄 기능 (3페이지 구성)

■ 설치 및 실행 방법
--------------------------------------------------------------------------------

1. 사전 요구사항
   - Node.js 18.x 이상 설치 필요
   - npm 또는 yarn 패키지 매니저

2. 의존성 설치
   > npm install

3. 개발 서버 실행
   > npm run dev
   
   브라우저에서 http://localhost:3000 접속

4. 프로덕션 빌드
   > npm run build
   > npm start


■ 평가 체계
--------------------------------------------------------------------------------

[기본 자격] - Pass/Fail 방식
  └─ Fail 항목이 있어도 평가는 진행되지만, 최종 결과는 Fail 처리

[기업 역량] - 40점 만점
  ├─ 재무 안정성 (23.3%)
  │   └─ 초기 기업: 택1 선택 (현금흐름/이자보상비율, 투자금액/투자기관)
  │   └─ 성장/성숙 기업: 투자 유치 실적 선택사항
  ├─ 운영 지속성 (33.3%)
  └─ 기술 신뢰성 (43.4%)

[공교육 가치] - 60점 만점
  ├─ 교육적 가치 (48.9%)
  │   ├─ 효과성 (42.2%)
  │   ├─ 적합성 (37.8%)
  │   └─ 혁신성 + 전문성 (20%)
  ├─ 기술적 가치 (24.4%)
  │   ├─ 공통 기술 역량 (60%)
  │   └─ BM 유형별 역량 (40%)
  └─ 사회적 가치 (26.7%)
      ├─ 사회적 책임 (32.6%)
      ├─ 윤리 준수 (34.1%)
      └─ 협력생태계 (33.3%)


■ 등급 기준 (10등급 체계, 4구간)
--------------------------------------------------------------------------------
  A 구간 (공교육 시장에서 즉시 활용 가능한 기업군)
    - AAA: 95~100점
    - AA: 90~95점 미만
    - A: 85~90점 미만

  B 구간 (공교육 시장 진출이 가능한 기업군)
    - BBB: 80~85점 미만
    - BB: 70~80점 미만
    - B: 65~70점 미만

  C 구간 (공교육 시장 진출을 위해 보완이 필요한 기업군)
    - CCC: 60~65점 미만
    - CC: 55~60점 미만
    - C: 50~55점 미만

  D 구간 (공교육 시장 진출이 어려운 기업군)
    - D: 50점 미만


■ 주요 특징
--------------------------------------------------------------------------------

1. 동적 평가 항목
   - 선택한 비즈니스 모델에 따라 관련 평가 항목만 표시
   - 기업 단계에 따른 평가 기준 차등 적용

2. 기업 단계별 맞춤 평가
   - 초기 기업: 재무 건전성/자금조달역량 "택1" 선택
     * 미선택 지표의 가중치는 해당 영역 내 다른 지표에 비례 배분
   - 성장/성숙 기업: 투자 유치 실적 선택사항
     * 체크박스로 평가 포함 여부 선택 가능
     * 선택 취소 시 같은 항목 다시 클릭

3. BM 연동 체크리스트
   - 학습데이터 활용, 핵심 스킬셋 보유 항목은 선택된 BM 유형별로 그룹화
   - 기술 성과물은 BM별 세부 평가 항목(스케일/체크리스트) 제공
   - 아코디언(접이식) UI로 체크리스트 그룹 관리

4. 하위 영역 구분
   - 학습경험 설계: 사용성/접근성/몰입형 설계 3개 영역
   - 기술 확장성: 시스템 연계 능력/데이터 이관 기능 2개 영역
   - 디지털 격차 완화: 저사양 기기/오프라인/데이터 절감 3개 영역

5. 상호 배타적 선택
   - 현장 활용도 평가 시 "도입기관 수" 또는 "누적 이용자 수" 중 택1

6. 모던 평가 결과 UI
   - 그라데이션 배경의 Hero 섹션 (등급/점수/기본자격)
   - SVG 기반 도넛 파이차트로 영역별 점수 시각화
   - 달성률(%)과 획득점수/만점 명확히 구분 표시
   - 영역별 구분된 종합 제언 자동 생성

7. 점수 일관성 보장
   - 세부 점수 합계 = 종합 점수 (반올림 오차 해결)
   - 체크리스트 점수 티어 방식 적용 (0~5점)

8. 사용자 편의성
   - 페이지 이동 시 자동 스크롤 (맨 위로)
   - 이전/다음 버튼 완전 동작
   - 체크리스트 아코디언 UI (그룹별 접기/펼치기)
   - 2열 그리드 체크리스트 레이아웃
   - 선택사항 항목 클릭으로 선택 취소 가능

9. 인쇄 기능
   - 3페이지 구성 PDF 인쇄 지원
   - 섹션별 페이지 나눔 최적화
   - 인쇄용 헤더/푸터 자동 표시
